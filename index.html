ActiveYelp é uma aplicação Full Stack baseada nas funcionalidades básicas do Yelp.

É um app responsivo, no qual os usuários podem pesquisar estabelecimentos destinados à prática esportiva de acordo com a região de seu interesse com o auxilio de um mapa interativo.

StockSimple Manager

Single Page Application que funciona como um sistema de gestão de estoque de uma empresa.

Backend com PHP, Laravel e MySQL.

Frontend com VueJS.

StockSimple Manager é um aplicação web do tipo single-page que permite realizar a gestão de estoque em uma empresa.

Entre as funcionalidades que o sistema possui para auxiliar no controle e organização de produtos, estão:

  • Cadastro de produtos: permite cadastrar, registrar e consultar informações detalhadas sobre os produtos, como nome, categoria, descrição, código, fornecedor, preço, quantidade em estoque, entre outros.
  • Controle de estoque: possibilita monitorar a entrada e saída de produtos, atualizando automaticamente o saldo disponível em estoque. Isso ajuda a evitar a falta de produtos ou o excesso de estoque.
  • Controle de vendas e registro de ordens: facilita o registro e controle das vendas realizadas por meio da geração de Ordens de Venda, atualizando automaticamente o saldo em estoque e fornecendo informações sobre o desempenho de vendas.
  • Cadastro de funcionários: viabiliza o cadastro de funcionários envolvidos na gestão de estoque. É possível também o realizar o registro de pagamento dos salários dos funcionários, e consultar esses registros de acordo com o mês de escolha.
  • Controle de despesas: com a possibilidade do registro de despesas relacionadas com o gerenciamento de estoque, e com os dados de saídas devido ao pagamento dos salários dos funcionários, tem-se a possibilidade de melhor análise de lucratividade.
  • Dashboard de análise: permite verificar rapidamente análises sobre o estoque e desempenho de vendas, como lista de produtos com estoque zerado, vendas realizadas e despesas até então no mês.
    O app pode ser facilmente modificado para proporcionar análises mais amplas de acordo com as necessidades de uma empresa, já que há uma grande variedade de dados que são coletados no banco de dados.
  • Sistema Point of Sale: O registro de ordens de venda é facilitado com um sistema de caixa/Point of Sale. Nesse sistema há fácil acesso aos produtos armazenados por categoria e simples preenchimento das informações da transação de venda para registro na ordem a ser emitida.
